# 擂椒皮蛋

- 青椒 若干
- 皮蛋 3个
- 蒜瓣 2
- 生抽

## 步骤
- 1 青椒洗净，油煎，煎软
- 2 皮蛋过开水
- 3 皮蛋和青椒放入蒜臼，加生抽蒜沫捣碎
